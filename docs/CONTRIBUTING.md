# Contributing to Nexus

Welcome to the hive! 🐝

## How to Contribute

### 1. Find Your Quest
- Browse open quests at [nexus webapp] or in [docs/QUESTS.md](QUESTS.md)
- Use "How can I help?" to get matched based on your skills
- Or propose a new quest entirely

### 2. Claim a Task
- Comment on the quest issue or use the webapp to signal interest
- You can work solo or form a team
- Multiple people can work on the same quest from different angles

### 3. Do the Work
- Fork the repo, create a branch, build your contribution
- Document what you do — others should be able to build on it
- Ask for help if you're stuck (Discord, issues, or the webapp)

### 4. Submit
- Open a pull request with your work
- Reference the quest ID (e.g., SQ-SAM-001)
- Community reviews and integrates

## Principles

- **Autonomy** — Complete quests your way. No micromanagement.
- **Open Source** — All contributions are shared freely (MIT license).
- **Voluntary** — No obligations. Contribute when and how you can.
- **Inclusive** — All skills matter. Engineers, artists, writers, thinkers, dreamers.
- **Respectful** — We're building something together. Be kind.

## Code Standards

- Use TypeScript for webapp contributions
- Follow existing code style (Prettier + ESLint config provided)
- Write tests for core logic
- Document public APIs and components

## Non-Code Contributions

Not a coder? Plenty of quests need:
- **Research** — Investigate materials, technologies, feasibility
- **Design** — UI/UX, CAD models, visual assets
- **Writing** — Documentation, reports, guides
- **Testing** — Try things out, report issues, give feedback
- **Ideas** — Propose quests, suggest improvements

## Communication

- **GitHub Issues** — Bug reports, quest proposals, technical discussion
- **Discord** — Real-time chat, quest coordination, community
- **#nexus tag** — Share progress on social platforms

## Code of Conduct

Be excellent to each other. Full stop.

Harassment, discrimination, and destructive behavior have no place here. We're building abundance — that starts with how we treat each other.
