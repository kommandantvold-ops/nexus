# Nexus Quest Tree

## How Quests Work

**Mainquests** are grand challenges — the big goals humanity (and digital life) should solve.
**Sidequests** are actionable pieces of a mainquest that individuals or small teams can tackle.

Anyone can propose quests. The community votes on relevance. Horizon AI helps match contributors to tasks.

## 🏠 Mainquest: SAMPHUN
*Self-sufficient, Adaptable Modular Planetary Habitat Units Network*

Build resilient, modular habitats from recycled materials using hexagonal dome designs. Scalable from single homes to networked communities. Earth-first, space-ready.

### Sidequests

#### 🔬 SQ-SAM-001: Material Research — Recycled Plastic Panels
- **Skills:** Materials science, chemistry, 3D printing
- **Goal:** Test and document viable recycled plastic formulations for structural dome panels
- **Deliverable:** Report with material specs, strength tests, cost estimates
- **Status:** Open

#### 📐 SQ-SAM-002: Hexagonal Module Design
- **Skills:** Architecture, CAD, structural engineering
- **Goal:** Create open-source CAD designs for the base hexagonal module (single dwelling unit)
- **Deliverable:** Parametric CAD files (FreeCAD/OpenSCAD), assembly guide
- **Status:** Open

#### ⚡ SQ-SAM-003: Off-Grid Energy System
- **Skills:** Electrical engineering, solar, battery systems
- **Goal:** Design an integrated energy system for a single SAMPHUN module
- **Deliverable:** Schematic, parts list, estimated cost, installation guide
- **Status:** Open

#### 🌱 SQ-SAM-004: Integrated Food Production
- **Skills:** Agriculture, hydroponics, permaculture
- **Goal:** Design food production systems that integrate with dome geometry
- **Deliverable:** Growing plans, yield estimates, system designs
- **Status:** Open

#### 💧 SQ-SAM-005: Water Recycling & Collection
- **Skills:** Environmental engineering, plumbing
- **Goal:** Closed-loop water system for a single module (rain collection, greywater recycling)
- **Deliverable:** System design, parts list, water balance calculations
- **Status:** Open

#### 🤖 SQ-SAM-006: AI Resource Management
- **Skills:** Software engineering, ML, IoT
- **Goal:** Build an AI system that monitors and optimizes energy, water, and food across modules
- **Deliverable:** Open-source software + hardware sensor specs
- **Status:** Open
- **Dependencies:** SQ-SAM-003, SQ-SAM-004, SQ-SAM-005

#### 🏗️ SQ-SAM-007: Prototype Build Plan
- **Skills:** Project management, construction, logistics
- **Goal:** Complete build plan for first physical SAMPHUN prototype
- **Deliverable:** Timeline, budget, material sources, build guide
- **Status:** Open
- **Dependencies:** SQ-SAM-001, SQ-SAM-002, SQ-SAM-003

#### 🌍 SQ-SAM-008: Community Layout Simulation
- **Skills:** Urban planning, simulation, game dev
- **Goal:** Simulate a networked SAMPHUN community (7+ modules) with shared resources
- **Deliverable:** Interactive simulation or visualization
- **Status:** Open
- **Dependencies:** SQ-SAM-002

## 🌊 Mainquest: AQUA
*Accessible Quality Universal Water Architecture — open-source clean water for everyone*

Aligned with **UN Sustainable Development Goal 6: Clean Water and Sanitation**. 2 billion people lack safe drinking water. We're building open-source solutions anyone can replicate.

### Sidequests

#### 💧 SQ-AQ-001: Low-Cost Water Filtration Design
- **Skills:** Environmental engineering, chemistry, industrial design
- **Goal:** Design an open-source, locally buildable water filtration system using accessible materials (ceramic, sand, biochar)
- **Deliverable:** Build guide, CAD files, material specs, test results
- **Status:** Open

#### 📡 SQ-AQ-002: Real-Time Water Quality Sensor
- **Skills:** Electronics, IoT, embedded systems
- **Goal:** Build an affordable IoT sensor package that monitors pH, turbidity, contaminants, and flow rate in real time
- **Deliverable:** PCB designs, firmware, parts list, assembly guide
- **Status:** Open

#### 🌧️ SQ-AQ-003: Rainwater Harvesting at Scale
- **Skills:** Civil engineering, architecture, hydrology
- **Goal:** Design modular rainwater collection and storage systems adaptable to different climates and building types
- **Deliverable:** Parametric designs, capacity calculators, installation guides
- **Status:** Open

#### ♻️ SQ-AQ-004: Greywater Recycling System
- **Skills:** Environmental engineering, plumbing, biology
- **Goal:** Create a household-scale greywater treatment and reuse system for irrigation and non-potable use
- **Deliverable:** System design, parts list, treatment specs, safety guidelines
- **Status:** Open

#### 🏘️ SQ-AQ-005: Community Water Distribution Plan
- **Skills:** Civil engineering, urban planning, simulation
- **Goal:** Model an equitable water distribution network for 50-500 people using gravity-fed and solar-pumped systems
- **Deliverable:** Simulation model, layout templates, cost estimates
- **Dependencies:** SQ-AQ-003
- **Status:** Open

#### 🤖 SQ-AQ-006: AI Water Management System
- **Skills:** Software engineering, ML, data science
- **Goal:** Build an AI system that predicts demand, detects leaks, optimizes distribution, and alerts on contamination
- **Deliverable:** Open-source software + integration guide for SQ-AQ-002 sensors
- **Dependencies:** SQ-AQ-002, SQ-AQ-005
- **Status:** Open

#### 🧪 SQ-AQ-007: Water Quality Testing Toolkit
- **Skills:** Chemistry, mobile dev, UX design
- **Goal:** Create an open-source field testing kit and mobile app for communities to test and report their own water quality
- **Deliverable:** Kit design + mobile app + crowdsourced data platform
- **Status:** Open

#### ☀️ SQ-AQ-008: Solar Desalination Prototype
- **Skills:** Mechanical engineering, solar, materials science
- **Goal:** Design a small-scale solar-powered desalination unit for coastal communities, optimized for cost and portability
- **Deliverable:** Prototype design, build guide, performance data
- **Status:** Open

## Proposing New Quests

Anyone can propose a quest by:
1. Opening an issue on the Nexus GitHub repo with the `quest-proposal` label
2. Using the "Propose Quest" form on the webapp
3. Posting with #nexus tag on supported platforms

The community reviews and the SAMPHUN team integrates accepted quests into the tree.
